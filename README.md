<pre># Console-Based To-Do List Application

A simple Console-Based To-Do List application built using Python.
This project allows users to add, view, and remove tasks through a command-line interface while storing tasks permanently in a text file.

## Features

* Add new tasks
* View all saved tasks
* Remove completed tasks
* Persistent task storage using text files
* User-friendly CLI interface
* Error handling for invalid inputs

## Technologies Used

* Python
* VS Code / Any Text Editor
* Command Line Interface (CLI)

## Concepts Used

* File Handling
* Lists
* Functions
* Loops
* Conditional Statements
* String Manipulation

## How to Run

1. Save the file as `todo.py`
2. Open terminal in the project folder
3. Run the command:

```bash id="ymc6hm"
python todo.py
```

## Outcome

This project demonstrates the fundamentals of Python file handling and helps beginners understand how to build persistent command-line applications.
</pre>
